Swift - Movie Ticket Booking System

“Swift” is a state-of-the-art movie ticket booking platform designed to streamline the entire booking process. The system integrates real-time updates for ticket availability, ensuring that users always receive accurate and timely information. I designed the system to handle high traffic during peak hours while ensuring the accuracy and integrity of bookings.

The biggest technical challenge was handling race conditions during ticket reservations. By implementing robust thread synchronization and utilizing advanced inventory management algorithms, I ensured that tickets were reserved without errors even during high-demand situations.

Tech Stack: Java, Spring Boot, ReactJS, MySQL

Microservices Architecture for Online Shopping App

This project involved designing and implementing a comprehensive microservices architecture for a scalable online shopping platform. The system was built to handle millions of concurrent users, with distinct services for product management, order processing, inventory control, and user notifications.

I integrated **Netflix Eureka** for service discovery, implemented **circuit breakers** for fault tolerance, and used **Apache Kafka** for real-time communication between services. These technologies ensured that the platform could scale efficiently while maintaining high availability and resilience during failures.

Tech Stack: Java, Spring Boot, Apache Kafka, Docker, MongoDB, MySQL
